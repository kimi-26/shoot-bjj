Google Drive 연동 설정

딱 한 번만 설정하면 학생들이 바로 Drive에 저장할 수 있어요.

필요한 것: Google Cloud Console에서 OAuth 클라이언트 ID 발급 (무료, 5분 소요)

1Google Cloud Console 접속

console.cloud.google.com → 새 프로젝트 → 이름 아무거나

2Drive API 사용 설정

APIs & Services → Library → "Google Drive API" → 사용 설정

3OAuth 동의 화면 설정

APIs & Services → OAuth consent screen → External → 앱 이름 입력 → 저장

4OAuth 클라이언트 ID 만들기

Credentials → Create Credentials → OAuth client ID → Web application
