Google Drive 연동 설정

Apps Script로 간단하게 연동해요. Google Cloud 설정 없이 가능해요!

1

Apps Script 열기

script.google.com 접속 → 새 프로젝트

2

코드 붙여넣기

기존 코드 전부 지우고 drive-server.gs 파일 내용 붙여넣기

← 이 파일은 카메라 앱과 같이 GitHub에 올려뒀어요

3

웹앱으로 배포

오른쪽 위 배포 → 새 배포
유형: 웹 앱
실행 계정: 나(Me)
액세스 권한: 모든 사용자(Anyone)
→ 배포 → URL 복사

4

URL 입력 후 저장
